# Edu_AI_Risk
# ADMINISTRACIÓN DEL DESARROLLO DE SOFTWARE 

Proyecto: EduAI Risk Docente
Predicción de riesgo académico con IA para apoyo docente

EduAI Risk Docente, una solución simplificada con enfoque de Inteligencia Artificial que busca apoyar a profesores en la identificación temprana de estudiantes con riesgo de reprobar una materia. 
El producto se basa en una base sintética de estudiantes, el entrenamiento de un modelo sencillo de clasificación y un dashboard docente con alertas de riesgo. Como complemento, se plantea un chatbot básico de orientación que entrega recomendaciones pedagógicas simples según el nivel de riesgo del estudiante.
